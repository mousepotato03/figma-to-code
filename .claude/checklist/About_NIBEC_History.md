## 페이지 개요

NIBEC 회사의 연혁(History) 페이지입니다. 2000년부터 2025년까지의 회사 주요 이정표를 타임라인 형식으로 보여주는 페이지로, 펩타이드 기술 기반 바이오 기업의 성장 과정을 연대순으로 정리하여 표시합니다.

## 구현 권장사항

### 레이아웃
- 전체 페이지 너비 1920px 기준, 컨텐츠 영역 1280px 중앙 정렬
- 타임라인 좌측 858px, 우측 이미지 갤러리 362px 분할
- 연도별 타임라인 아이템 간 간격 일정하게 유지

### 반응형 고려사항
- 모바일: 이미지 갤러리를 타임라인 아래로 이동 또는 숨김 처리
- 타블릿: 2컬럼 유지하되 너비 비율 조정
- 타임라인 수직선과 도트 위치 반응형 조정 필요

### 접근성
- 연도별 섹션에 적절한 heading 태그 사용 (h2, h3)
- 타임라인 이벤트 목록에 시맨틱 리스트 마크업
- 이미지에 적절한 alt 텍스트 제공

### 인터랙션
- 스크롤 시 현재 연도 하이라이트 효과 고려
- 연도 클릭 시 해당 섹션으로 스무스 스크롤
- 이미지 갤러리 라이트박스 또는 확대 기능 고려
